# PromptG

This repo includes a `.promptg/` directory that stores prompts in the **PromptG** standard format, so they can be versioned alongside the codebase and used consistently by different tools.

- Website: `https://promptg.io`
- CLI: `https://github.com/promptg/cli`

## Layout

- Prompts live in `.promptg/prompts/`
- Filenames follow the spec convention: `promptg-prompt-{name}.json`
